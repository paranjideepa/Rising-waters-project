# 🌊 Rising Waters - Flood Prediction System

## 📌 Project Overview

The Rising Waters Flood Prediction System is a Machine Learning project developed to predict the possibility of floods based on rainfall and weather-related parameters. The project helps demonstrate how machine learning can be applied to disaster prediction and early warning systems.

---

## 🎯 Objectives

- Analyze flood-related data.
- Preprocess the dataset.
- Train multiple Machine Learning models.
- Compare model performance.
- Build a web application using Flask for prediction.

---

## 📂 Dataset

Dataset Used:
- flood dataset.xlsx

Features:
- Temperature
- Humidity
- Cloud Cover
- Annual Rainfall
- Jan-Feb Rainfall
- Mar-May Rainfall
- Jun-Sep Rainfall
- Oct-Dec Rainfall
- Average June Rainfall
- Sub Rainfall

Target:
- Flood (0 = No Flood, 1 = Flood)

## 🚀 How to Run

1. Clone the repository.
2. Install the required libraries:

```
pip install -r requirements.txt
```

3. Run the application:

```
python app.py
```

4. Open your browser:

```
http://127.0.0.1:5000
```
